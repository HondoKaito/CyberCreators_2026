◆このゲームについて
　BAYONETTA3とバトルモービルをモチーフにしたアクションシューティングゲーム！
　NINJAを操作して追いかけてくる怪物『バブリー』を撃破し、研究施設から脱出しよう！
　

◆起動方法
exeファイルを実行する。


◆基本操作
Wキー:奥に移動

Aキー:左移動

Sキー:手前に移動

Dキー:右移動

SPACEキー:ジャンプ

Kキー:通常弾（苦無）発射
・通常弾

Jキー:後方に爆弾投擲
・後ろに爆弾を投げる

Lキー:高速移動
・高速移動で回避する

◆エネミー
バブリー
・研究施設の人間によって作られた生体兵器。脱走してプレイヤーに襲い掛かる
・変身をあと二回残している

子バブリー
・バブリーが放つ弾。頭に爆弾を乗せて特攻してくる

バブリーレッグ
・バブリー第三形態が使用する攻撃、ジャンプ回避でよけるか爆弾で破壊しよう

ゴキゲン・ミ゛ーミ゛ー・大罪
・研究所で造られている生体兵器たち。バブリーとの戦いの最中に物理的に視界を見えにくくする

◆実装内容
